Freshmail
==========

Module integrate drupal 8 with Freshmail mailing system - http://freshmail.pl/

Full project site - https://www.drupal.org/project/freshmail

CONFIGURATION
=============
Configure page - /admin/config/content/freshmail-settings
You need add list hash, API key, API secret key.


Freshmail block
=================
Create block, where user can add email to newsletter.
